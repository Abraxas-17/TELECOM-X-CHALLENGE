# TELECOM-X-CHALLENGE
CHALLENGE TELECOM X
Análisis de Evasión de Clientes (Churn) – Telecomunicaciones
📌 Descripción del Proyecto

Este proyecto analiza la evasión de clientes (Churn) en una empresa de telecomunicaciones con el objetivo de identificar patrones y factores que influyen en la cancelación del servicio.
Mediante técnicas de análisis exploratorio de datos (EDA) se examinan variables demográficas, contractuales y de consumo para detectar comportamientos asociados al abandono del servicio.
El análisis busca generar insights que ayuden a mejorar la retención de clientes y apoyar la toma de decisiones estratégicas.

🎯 Objetivos

Analizar los factores asociados al churn de clientes.
Identificar patrones en variables numéricas y categóricas.
Detectar segmentos de clientes con mayor riesgo de cancelación.
Generar recomendaciones estratégicas basadas en datos.

🛠️ Herramientas Utilizadas

Python
Pandas – manipulación y análisis de datos
Matplotlib / Seaborn – visualización de datos
Jupyter Notebook – desarrollo del análisis

📊 Principales Hallazgos

El análisis exploratorio reveló varios factores asociados con la cancelación del servicio:
Los clientes con contratos mensuales (Month-to-month) presentan mayor tasa de churn.
Los clientes con menor tiempo de permanencia (tenure) tienen mayor probabilidad de cancelar.
El método de pago Electronic Check está asociado con tasas más altas de evasión.
Los clientes con servicio de fibra óptica presentan mayor tasa de cancelación.
El género no mostró impacto significativo en el churn.

💡 Recomendaciones

Incentivar contratos de mayor duración para aumentar la retención.
Promover el uso de métodos de pago automáticos.
Implementar estrategias de retención en los primeros meses del cliente.
Analizar posibles problemas en el servicio de fibra óptica.
